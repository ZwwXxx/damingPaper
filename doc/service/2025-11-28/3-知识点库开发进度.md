
2. **创建Controller层**
   ```bash
   # 学生端: 查询知识点、查看详情、增加浏览量
   # 管理端: CRUD操作、审核管理
   ```

3. **启动测试**
   ```bash
   # 启动后端项目
   cd daming-admin
   mvn clean package
   java -jar ruoyi-admin/target/ruoyi-admin.jar
   
   # 访问接口测试
   # GET http://localhost:8080/student/knowledge/points
   ```
