# 大明在线考试系统项目简介

## 1. 项目概述

### 1.1 项目背景
随着教育信息化的快速发展和在线教育的普及，传统的纸质考试方式已无法满足现代教育的需求。为了提高考试效率、降低人工成本、增强数据统计分析能力，急需一套功能完善的在线考试系统。

### 1.2 项目目标
- **提高考试效率**：支持大规模在线考试，自动批阅客观题，提升考试组织效率
- **降低管理成本**：减少纸质试卷印刷、分发、回收等环节的人力物力成本
- **增强防作弊**：提供多种防作弊措施，保证考试公平性
- **数据分析**：提供详细的考试数据统计和分析功能，辅助教学改进

### 1.3 系统定位
本系统是一个面向教育机构的B/S架构在线考试平台，支持多角色用户使用，包括系统管理员、教师和学生。系统提供题库管理、试卷组卷、在线考试、成绩统计等全流程功能。

## 2. 技术架构

### 2.1 总体架构
系统采用前后端分离的三层架构设计：
- **展示层**：基于Vue.js 2.x + Element UI的前端应用
- **业务层**：基于Spring Boot 2.x的后端服务
- **数据层**：MySQL数据库 + MyBatis持久化框架

### 2.2 技术栈详情

#### 后端技术栈
- **核心框架**：Spring Boot 2.x
- **安全框架**：Spring Security + JWT
- **持久层框架**：MyBatis + MyBatis-Plus
- **数据库**：MySQL 5.7+
- **连接池**：Druid
- **缓存**：Redis
- **工具库**：Hutool、Apache Commons、Fastjson2

#### 前端技术栈
- **前端框架**：Vue.js 2.x
- **UI组件库**：Element UI
- **HTTP客户端**：Axios
- **构建工具**：Webpack + Vue CLI
- **样式处理**：SCSS

#### 开发工具
- **IDE**：IntelliJ IDEA
- **版本控制**：Git
- **接口文档**：Swagger
- **项目管理**：Maven

## 3. 功能模块

### 3.1 用户权限管理
- **角色管理**：系统管理员、教师、学生三种角色
- **权限控制**：基于RBAC模型的细粒度权限控制
- **用户管理**：用户注册、登录、信息维护
- **组织管理**：支持多级组织架构管理

### 3.2 题库管理模块
- **题目录入**：支持单选、多选、判断、填空、主观题五种题型
- **批量导入**：支持Excel格式的题目批量导入导出
- **题目编辑**：可视化题目编辑器，支持富文本内容
- **分类管理**：按科目、难度、知识点等维度分类管理
- **题目审核**：多级审核机制确保题目质量

### 3.4 考试管理模块
- **考试发布**：设置考试参数，发布考试通知

## 4. 数据库设计

### 4.1 核心数据表

#### 用户相关表
- `sys_user`：用户基本信息表
- `sys_role`：角色信息表
- `sys_user_role`：用户角色关联表

#### 题库相关表
- `daming_question`：题目基本信息表
- `daming_content_info`：题目详细内容表（JSON格式存储）
- `daming_subject`：科目管理表

#### 试卷相关表
- `daming_paper`：试卷基本信息表
- `daming_paper_answer`：考试答题记录表
- `daming_question_answer`：题目答题详情表

### 4.2 数据库设计特点
- **JSON存储**：题目内容采用JSON格式存储，灵活支持各种题型
- **内容分离**：题目基本信息与详细内容分表存储，提高查询效率
- **软删除**：重要数据采用软删除机制，保证数据安全
- **索引优化**：关键字段建立索引，优化查询性能

## 5. 系统特色功能

### 5.1 灵活的题目管理
- **多题型支持**：支持单选、多选、判断、填空、主观题
- **富文本编辑**：题目内容支持图片、公式、表格等富媒体
- **批量操作**：支持题目的批量导入、导出、删除等操作
- **版本管理**：题目修改历史记录，支持版本回退

### 5.2 智能组卷系统
- **规则配置**：可配置各题型数量、分值、难度比例
- **去重机制**：确保同一次组卷不出现重复题目
- **均衡分布**：自动均衡各知识点题目分布
- **预览确认**：组卷结果可预览，不满意可重新生成

### 5.3 防作弊机制
- **题目乱序**：同一试卷不同学生题目顺序不同
- **选项乱序**：选择题选项顺序随机打乱
- **时间控制**：严格控制考试时间，超时自动提交
- **环境检测**：检测切屏、复制粘贴等异常行为

### 5.4 数据分析功能
- **实时监控**：考试过程实时数据监控
- **多维统计**：按时间、科目、班级等多维度统计
- **趋势分析**：历史数据趋势分析和预测
- **报表生成**：自动生成各类分析报表
